# - Curso de JavaScript e TypeScript -

Este repositório é destinado aos **estudos, práticas e projetos desenvolvidos durante o curso de JavaScript e TypeScript**.  
O objetivo é consolidar os conhecimentos por meio de **exercícios práticos, exemplos de código, desafios e projetos**, acompanhando a evolução no aprendizado em desenvolvimento web.

---

## Objetivos do Repositório

- Aprender e reforçar os **fundamentos do JavaScript**
- Evoluir para conceitos **intermediários e avançados**
- Introduzir e aplicar **TypeScript**
- Praticar lógica de programação
- Desenvolver pequenos projetos práticos
- Manter um repositório organizado para **consulta futura e portfólio**

---

## Conteúdos Abordados

### JavaScript
- Variáveis e tipos de dados
- Operadores e estruturas de controle
- Funções
- Arrays e objetos
- Laços de repetição
- Manipulação do DOM
- Eventos
- Assincronismo (Promises, Async/Await)
- Boas práticas

### TypeScript
- Tipagem estática
- Interfaces e Types
- Funções tipadas
- Classes
- Enum
- Generics
- Integração com JavaScript
- Boas práticas com TypeScript
